# Local Macro Recorder (PC 插件)

## 目标
在 14 天内交付可演示、可上架 Chrome Web Store 的最小可用版本（MVP），解决 C 端用户高频重复网页操作问题。

## ICP
- 求职者/报名用户：重复填表
- 个体卖家/运营：后台重复操作
- 信息整理用户：重复网页提取

## 定位与定价
- 定位：**本地优先（local-first）** 的轻量浏览器宏录制器
- Free：3 个宏、基础录制/回放
- Pro：$3.99/月（或 $39/年）

## MVP 边界（冻结）
1. 录制：click/type/scroll/wait
2. 回放：同域重放 + 简单失败重试
3. 变量：CSV 批量填表
4. 日志：执行日志与失败原因
5. 模板：3 个（批量填表/抓取到CSV/固定后台流程）

## 里程碑
- D1：PRD+任务拆解+埋点定义
- D3：Alpha（录制/回放/变量）
- D7：Beta（日志+重试+3模板）
- D14：上架验证（安装/激活/付费）

## 非目标
- 不做跨应用自动化平台
- 不做复杂云端编排
- 不绕过验证码/安全机制
